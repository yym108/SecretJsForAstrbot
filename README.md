# AstrBot 街机厅人数计数插件

## 简介

当用户发送的消息中包含指定前缀时，机器人会自动开启计数功能。

## 功能

*   **关键词触发：** 自动检测消息中是否包含指定前缀。
*   **自动回复：** 当检测到关键词时，自动回复指定文本，并且记录下人数。

## 使用方法

1. 直接设定目前的人数，检测消息格式为".润='人数'
2. 人数增加或减少，检测消息格式为".+（或-）'人数'"
3. 询问目前人数。检测消息格式为".润几"

## 配置

本插件无需任何配置。

## 注意事项

*   本插件需要 AstrBot 机器人环境。
*   请确保 AstrBot 机器人已正确安装并配置。
